Welcome to fah-alchemy’s documentation!¶

fah-alchemy is a package for running large scale free energy campaigns on the Folding at Home (FAH) distributed computing platform. It is designed for maximum interoperability with the Open Molecular Software Foundation stack, and is built on top of OpenMM, OpenForceField and OpenFreeEnergy.

Note

This software is pre-alpha and under active development. It is not yet ready for production use and the API is liable to change rapidly at any time.
